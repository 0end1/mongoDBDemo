# mongoDBDemo
一个用Mongo，Node 实现增删改查的小应用


### 即将用到的技术:

- Angular 5+ & Typescript
- Bootstrap 4+ & SCSS
- Responsive layout
- High resolution
- Flexibly configurable themes with **hot-reload** (2 themes included)
- Authentication module with multiple providers
- Lots of awesome features:
  - Buttons
  - Modals
  - Popovers
  - Icons
  - Typography
  - Animated searches
  - Forms
  - Tabs
  - Notifications
  - Tables
  - Maps
  - Charts
  - Editors
  
And many more!
